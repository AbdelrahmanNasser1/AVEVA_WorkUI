Building XML Schema

You can build a XML Schema in two ways:

  1. Define schema in the XML Variable interface
    1. Adding a Node
    2. Editing a Node
    3. Deleting a Node
  2. Loading schema from an existing file

Defining Schema in the XML Variable Interface

When a new variable is created, the schema displays the root node by default. Click the string "root"and type the name over it.

To add a node

  1. To add a node, click the Properties icon to the right of the name. Select Add. The Properties window for the new node appears.
  2. Specify the following properties of the node:
    • Name: Enter a name of the node.
    • Type: Select the node type. The options available are element and attribute.
    • Data Type: Select the data type. This property specifies the data type of the XML Node. The available data types are string, integer, long, double, decimal, date, time, dateTime, boolean, and base64binary.
    • Repeating: Select this check box, if the node is a repeating one.
  3. Click OK.
  4. Click Save Variable to save node properties.
  5. Click OK.
  6. To add more nodes, repeat step 1 to step 6.

     

    Note: Select the respective node where you want add a node. If a node is of Attribute type, then you cannot add a node under it. The Add option will be disabled for Attribute type of node.

To edit an existing node

  1. To edit an existing node, click the Properties icon to the right of the node to be edited.
  2. Select Edit. The Properties window for the existing node appears.

  3. Modify properties in the Properties window.
  4. Click OK.

  5. Click Save Variable to save node properties.
  6. Click OK.

To delete a node

  1. To delete a node, click the Properties icon to the right of the node to be deleted.
  2. Select Delete

Loading Schema from an Existing File

  1. Click Browse to select file with schema information. You can select either InfoPath forms or XML files to load schema

  2. The XML schema is loaded from the file.
  3. To edit a node, right-click the node, and then select Edit. Modify properties in the Properties window.
  4. To delete a node, right-click the node, and then select Delete
