End Child Workflow: Workflow Details

Workflow Details property window

This topic gives you a detailed understanding of the Workflow Details properties. The Workflow Details properties are used to specify

  • the child workflow to be ended.
  • the workflow activity to which the child workflow that is to be ended is associated.

To open the Workflow Details window, click the Workflow Details edit button.

This window has the following properties:

Select Workflow/Activity: Select the name of the child workflow that is to be ended or the  name of the activity that is associated to the child workflow. Select the Child Workflow Name or Workflow Activity Name radio button.

Property Type: Mandatory (This property must be set.)

  • Child Workflow Name: Select the name of the child workflow that needs to be ended from the drop-down list. This option is enabled when you have selected the Child Workflow Name radio button in the Select Workflow/Activity property.

    Note: You can view the child workflow in the drop-down list only if you have configured the child workflow in the Workflow activity.

  • Specify Workflow Name: Select this option and type the name of the child workflow in the adjacent field. Alternatively, you can click the expression editor icon and select a variable assigned to the child workflow in the expression editor window.
  • Workflow Activity Name: Select the name of the workflow activity used in the parent workflow, to which the child workflow is associated. The workflow activity name can be selected from the drop-down list. This option is enabled when you have selected the Workflow Activity Name radio button in the Select Workflow/Activity property.
  • Specify Activity Name: Select this option and type the name of the workflow activity to which the child workflow is associated, in the adjacent field. Alternatively, you can click  the expression editor icon and select a variable assigned to the workflow activity in the expression editor window.
