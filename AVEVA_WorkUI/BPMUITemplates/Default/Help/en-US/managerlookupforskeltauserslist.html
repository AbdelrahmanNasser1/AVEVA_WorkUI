Manager Lookup for Users List

When creating a new user in AVEVA Work Tasks List, AVEVA Work Tasks had options for Name, Password, Description, Email, Department, and Role options. Now AVEVA Work Tasks introduced a new option to even select the Manager for that new user, where the Manager Lookup is a self lookup.

By default, the admin user is created when a new repository is created. For all the newly created user, you can select the existing users as a manager.

The following figure shows that, there are totally four users are existing at repository. Using the Manager lookup, any existing user can be assigned as manager for a user.
