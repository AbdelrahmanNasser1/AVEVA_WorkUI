KPI(s)

You can use the KPI(s) tab to define thresholds for the KPI data points identified in the previous tab.

To add a KPI:

  1. Click New KPI. A dialog box appears.
  2. Enter the following information:
    1. Name: Name for the new KPI.
    2. Data Point(s): Click Add New to add a Data Point, and enter the overall Threshold for it.

       
  3. Slots for Trend Analysis: Select the time periods for trend analysis.
  4. Click Save.
