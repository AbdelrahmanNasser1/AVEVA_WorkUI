Retrieving Node Values

Sending Employee information through email to a manager.

Solution: In this scenario, to display Employee Name, select the Employee Name node and to get Department, select Display Text node. Use Email activity. Set Body(Memo text) property as depicts in the following table.

Note: For string values, the Display Text node will not be displayed. In this case, you need to select SiteColumn name.

Field Name

Display Text

Employee Name

<%#XmlVariables.Employee.List.ListItem.SiteColumns.Employee Name%> (For string value)

Department

<%#XmlVariables.Employee.List.ListItem.SiteColumns.Department.DisplayText%> (For different Datatypes other than String, you need to select Display Text node)
