Disable Reply Email for Invalid Emails

Reply emails for invalid mails can be disabled. If any invalid email comes to the notification engine, an automated reply – "System could not process your mail. Some of the keywords not found in your mail to process it. Please reply to the original message again" will be send to the from address with the same subject. This action has been made configurable now. User can disable this reply if required.

This can be achieved by adding a registry string value called SendReplyForInvalidMail under HKEY_LOCAL_MACHINE\SOFTWARE\Skelta\BPM.NET and set its value data as NO.
