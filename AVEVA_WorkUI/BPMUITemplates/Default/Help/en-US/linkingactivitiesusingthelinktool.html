Linking Activities Using the Link Tool

Another method for linking two activities is to use the Link tool available in the Tool bar of the Process Designer.

To link two activities in this way:

  1. Click the Link tool in the tool bar.
  2. In the Link Actions window that opens, specify the first activity to be linked in the From Activity field.
  3. Next specify the second activity to be linked in the To Activity field.
  4. Click OK.

    This will add a link connecting the two activities in the design mosaic.

See also: Link Actions.
