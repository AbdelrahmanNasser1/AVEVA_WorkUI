Placeholder

Activity description:

The Placeholder Activity is used as a placeholder for the BPMN events.

Activity Properties:

The Placeholder activity has to be configured by specifying appropriate values for the different properties in the Activity Properties area. The Activity Properties can be accessed by clicking the Activity Properties tool in the tool bar or by selecting the appropriate option from the context menu for the Placeholder activity. The Activity Properties are organized under the following groups of related properties. The properties under each group are described in this topic.

Name & Description

The properties in the Name & Description group have been described below. You can use these properties to specify the name and description for the activity.

Name - This property can be used to specify a name for the activity.

Property Type: Optional (This property needs to be set only if necessary.)

Description - This property specifies a brief description to be displayed for the activity.

Property Type: Optional (This property needs to be set only if necessary.)

Miscellaneous

The properties in the Miscellaneous group have been described below. You can use these properties to specify the web service details.

Action Output - This property allows the user to specify the action output(s) for the BPMN Placeholder activity.

Property Type: Mandatory (This property must be set if the activity is to be executed.)

Steps to set the Action Outputs property

  1. To specify the action output, click button for the 'Action Output' property to open the property window.
  2. In this window, enter the action output value in the Output String field.
  3. After entering the details, click Add Row to add this action output setting and display a new row.
  4. Remove rows if necessary by selecting them and clicking Remove Selected.
  5. After setting the action output settings, click Update.

See Placeholder - Action Outputs for a detailed description of the Action Outputs property window.

Activity outputs:

The Intermediate Event Placeholder Activity does not return any value.

Exception Behavior

Each activity has the Raise Error on Unhandled Linked Output property on click of which, it shows all the default configured mapped error outputs in red with their check boxes selected.

See Mapped Error Output for more details.
