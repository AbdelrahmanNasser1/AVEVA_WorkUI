Importing Process Definition from Visio

This window opens when you select the Import from Visio option from the Tools menu.

Import Visio Process Definition window

You can use the following controls to import a process definition created with the ITP-Commerce Process Modeler for Visio.

  • Select a valid XML file created with ITP-Commerce Process Modeler - You can enter the name of a valid XML Process definition file from the Visio stencil in this field or click Browse to select the XML file.
  • Import - After selecting a Visio process definition XML file, you can click this button to import it.
  • Cancel - You can click this button to cancel the selection and close this window.

    Note: The import routine maps Visio BPMN activities to native AVEVA Work Tasks activities. You should configure the necessary properties after import.
