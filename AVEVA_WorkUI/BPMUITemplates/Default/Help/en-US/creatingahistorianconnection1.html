Creating a Historian Connection

You can configure the Historian Connection information in the Database Connections Settings in the Enterprise Console.

  1. On the Enterprise Console, select Settings, and then click Database Connections. The Database Connection page appears. This page lists the Connection List. 
  2. Click New. The New Connection dialog box appears.

  3. Enter the details as shown in the image given below: 

  4. Then click Save. The database connection is saved and will appear in the Connection List page.
