Using Lists Inside InTouch

Follow Step 1 to Step 7 listed in the Initial Steps and continue:

  1. Select the control in the editor and double click on Lists Configuration under Animations. This will open up a new configuration page from where the Lists control can be configured. After configuring the Lists, click OK. Save and close the Editor.

  2. Select the instance and deploy in the ArchestrA IDE.
  3. Double-click the $InTouchViewApp instance (aaReactorDemo) under ViewEngine from the ArchestrA IDE.

    The InTouch WindowMaker appears.

  4. Create a new window for the Lists control.

  5. Provide the name and size for the window.

  6. Right-click inside the newly opened window and select the Embed ArchestrA Graphic option.

    This opens up the ArchestrA Galaxy Browser.

  7. Select the Instances tab and select the object to which the graphic symbol has been added.
  8. Select the name of the graphic object (Lists) to be displayed.

  9. Enable security for the Lists window, if required.

  10. Save the window and exit InTouch WindowMaker.
  11. Save and close the object in the ArchestrA IDE.
  12. Deploy the $InTouchViewApp instance.
  13. Open the InTouch Viewer and navigate to Special-> Security->Log On.
  14. Enter the credentials and click OK.

  15. Open the created Lists window from the File menu.

  16. The Lists Control loads in the window.
