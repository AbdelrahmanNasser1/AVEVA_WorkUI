Storing the Instant Messenger ID for the User

Notification messages can be sent to the Users Instant Messenger Id which is specified in the User List. The section below gives details of storing the IM Id for a User in AVEVA Work Tasks List provider.

  1. Log on to the Enterprise Console.
  2. Click Settings, and then click Users.

     
  3. Select the AVEVA Work Tasks user to whom you want to send the notification. Set the IM Id field of the selected user with the destination Messenger Id like destination@hotmail.com.

    This IM Id is used to send notification to the user.
