Using Web Parts Infrastructure

The Enterprise Console provides a rich set of tools and user controls to manage, control workflows, users. All the AVEVA Work Tasks Controls can be accessed inside the Enterprise console based on the security rights.

Personalization of the controls play an important role in many of the scenarios where in which user can configure their configuration or settings for the control overriding the global or public configuration or view.

And also in many scenarios, there is a requirement of setting the customized home page for the application site where the AVEVA Work Tasks controls should be able to place inside a specific layout and configure properties for the same.
