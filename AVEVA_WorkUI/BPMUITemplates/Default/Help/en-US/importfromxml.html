Import from XML

You can import process designs saved as valid XML files. This can be used to import and modify a process design to create a new one.

To import a process definition XML file:

  1. From the Tools menu, select the Import option.
  2. In the window that opens, enter the name of the XML Process definition file in the field provided. Alternatively, click on the Browse button to select a valid process definition XML file.
  3. After selecting a process definition XML file, click on the Import button to import it.

This imports the selected XML process definition.
