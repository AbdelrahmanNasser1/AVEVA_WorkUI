Approval Activity Email Template

AVEVA Work Tasks Engine accesses the approval activity's email template from the following physical path:

[AVEVA Work Tasks Installed Path]\WorkflowElements\Default\en-US\Templates\OutlookEmailTemplate\ApprovalEmail.html

This template can be modified by the user as per their UI requirement. The mails received by the actor is based on the above mentioned mail template.

The following is a default email template text for the ApprovalEmail.html file:
