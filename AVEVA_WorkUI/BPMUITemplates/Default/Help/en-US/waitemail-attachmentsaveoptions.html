Wait for Email: Attachment Save Options

You can set the options to save the email attachments in this property window. To open this window, click the button for the 'Attachment Save Options' property.

In user specified location, you have to append the mail id i.e, if user specified location is "C:\SkeltaMailAttachements\", then actual location will be "C:\SkeltaMailAttachements\<mailId>" .

Example

Location : c:\Temp\

Actual: c:\Temp\123 –(123 is last mailid )

This window has the following fields to specify the pattern to save attachments.

  • The attachment drop-down field can be used to specify the attachment option by selecting the appropriate property from the drop-down field. You can choose to save all attachment files, specific files or file contents.
  • The value type drop-down field can be used to specify the value type to assign to the option selected in the attachment drop-down field.
  • The value text box/ drop-down field can be used to specify the pattern which matches with the "File Name" or "File Content" property selected in the attachment drop-down field. The control displayed here depends on the option selected in the value drop-down field. If you select "Value", a text box is displayed here, where you can enter the pattern directly. If you select Variable or Content, then a drop-down is displayed with a list of related collections. You can select the appropriate option from this drop-down field.

There are two buttons at the bottom of the screen.

  • Update - You can click this button to save the newly added pattern settings.
  • Close - You can click this to close the property window without saving the newly added settings.
