Placeholder: Action Outputs

You can set the Action Outputs settings in this property window. To open this window, click the button for the 'Action Outputs' property. It displays the timer settings in a grid, which also has a blank row where you can enter the settings for new Action Outputs.

The window has two tabs at the top to manage the rows of Action Outputs settings in the grid.

  • Add Row - You can click this tab to add the newly entered settings and display a new row to the grid where you can add another setting.
  • Remove Selected - You can click this tab to remove the Action Outputs settings selected in the grid.

The grid has the following parameters that you need to specify for the new Action Outputs setting.

  • Action Outputs - You can enter the output message for this activity in this field.

There are two buttons and a check box at the bottom of the screen.

  • Update - You can click this button to save the newly added Action Outputs settings.
  • Close - You can click this to close the property window without saving the newly added Action Outputs settings.
  • Last Row needs to be added - You need to select this check box to save the last row added.
