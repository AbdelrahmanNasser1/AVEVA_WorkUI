Variables

You can select the variables or XMLvariables and set the values to them in this property window. To open this window, click the button for the 'Set Variable' property.

The following window shows the Update Variable Property window:

Steps to update a variable:To assign values to variables, click the button for the 'Set Variable' property to open the property window.

  1. In this window, select the variable to which you want to assign or update the value.
  2. Depending on your selection on the left hand side, the value can be specified directly in a text box, or as options in a drop down. You can select a variable collection, content collection, XML Variables collection, or build an expression. In the case of Variables, you will need to select the specific XML element from the schema. For example, select 'Name' on the left hand side.

  3. After selecting variable, the following screen is displayed to enter the value.

  4. Enter the value for the variable in the text box. For example, enter name as 'John' as shown in the following figure.
    The window shown in the image below is used to Update the Variables and their values.

  5. If you want to create more variables, then click on Select to add variables.
  6. After setting the value for the variables, click OK button to save the settings.

    Note: Follow the same steps for updating XMLVariable.

See Forms XmlVariables for more information.

Note: For number data types, the minimum and maximum values are not validated. Enter only valid numeric values for number data types.
For numeric, DateTime, Date, or Time data types, ensure that the respective value is based on the culture set.
You can enter the input or use the Date and Time picker to enter the required date and time.
Date.CurrentDate is not supported. Use the tree view option to pick the required date and time.
The time format supports long data type to enable you to enter seconds.
