Wait for Event: Workflow Event Data XML Variable

You can create a new event variable in this property window. To open this window, click the edit button for the 'Event XML Data Variable' property.

To create event data XML variable:

  1. To open this window, click create a new variable for event data edit button.
  2. Enter the variable name in the text box.

  3. Click OK.
  4. Click Save in the Activity Properties window.
