Invoke Form - Mail Integration Scenario

AVEVA Work Tasks provides a powerful feature to integrate the invoked form in a mail. This is useful for user to get a direct link to AVEVA Work Tasks Form as an email. Without logging into Enterprise Console, the user can click the link that will open Forms in a browser.

Pre-requisites

  1. To configure your machine IP address for  receiving emails, see Configuring Farm Configuration Wizard for Receiving Emails.
  2. The Send Notification Email property should be set to Yes for this activity.

Scenario Description:

Consider a scenario where the manager has to approve or reject the leave application, sent by the subordinates. Where the manager requires direct link to access this leave application form which is filled by the subordinate as an email.

Scenario Requirements:

Design a form called 'Leave Form' with following controls and invoke this form using Invoke Form activity.

  • Employee Name (Text Input control)
  • From Date (Date Input control)
  • To Date (Date Input control)
  • Reason (Text Input Control)
  • Grid
  • BaseForm
  • Approve (Button)
  • Reject (Button)

There are two ways to send notification mail:

  1. Using Body property
  2. Using Template Path property
