Undo and Redo Actions

The Process Designer is very user-friendly and simplifies the task of designing of process diagrams. The Undo and Redo options are one of the many new features that are helpful to the designers.

  • To undo any action, select Undo from the Edit menu. For example, you can undo an accidental deletion of an activity.
  • To redo any action, select Redo from the Edit menu. For example, you may need to add two Information activities after an Approval activity to inform the user submitting a request, about its approval or rejection. After adding the first Information activity you can select Redo to add the second Information activity.
