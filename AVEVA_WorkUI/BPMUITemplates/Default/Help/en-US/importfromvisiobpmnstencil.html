Import from Visio (BPMN Stencil)

You can import a process definition created with the ITP-Commerce Process Modeler for Visio.

To import a Visio process model:

  1. From the Tools menu, select the Import from Visio Stencil.
  2. In the window that opens, enter the name of a valid XML process definition file from the Visio stencil in the field provided. Alternatively, click on the Browse button to select the XML file.
  3. After selecting a Visio process definition XML file, click on the Import button.
  4. This will import the Visio process definition into AVEVA Work Tasks.

See also, Import from Visio stencil

Refer to the topic Overview of Visio Process Import under chapter Importing BPMN Process Models from Microsoft Visio in the Developer Guide.
