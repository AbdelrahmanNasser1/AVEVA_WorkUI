Wait for Event: List ID

You can select the list name and list item for which the workflow has to be associated in this property window. To open this window, click the edit button for the 'List ID' property.

The window has following fields:

List Name - This allows you to select list name from the drop-down list.

Submit - Clicking this will save the List ID and List Item ID in the respective properties.

Close - Clicking this will close the window.

To set the List ID or List Item ID:

  1. Select event scope as 'List', if you want only List Id.
  2. Click Save in the Activity Properties window.
  3. Click the edit button to open the List ID property window. The following screen is displayed.

  4. Select list name from the drop-down list.
  5. Click Submit to display list id of the selected list in the List ID property.
  6. The List Id in the List ID fields respectively.
  7. Select event scope as 'ListItem', if you want both List Id and List Item Id.
  8. Click Save in the Activity Properties window.
  9. Click the edit button to open the List ID property window.
  10. Select list name from the drop-down list. For example, select Users from the List Name.
  11. After selecting list name, List Item field will be displayed.
  12. Select List item from the List item drop-down list. For example, select 'George' user from the available users.
  13. Click Submit.
  14. The List Id and List Item Id will be displayed in the List ID and List Item ID fields respectively.
