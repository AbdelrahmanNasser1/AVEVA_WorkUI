Invoke Form - Form Name

Form Name Property Window

To open this window, click the  button in the 'Form Name' property.

Form Name Properties window for Invoke Form

The Form Name window has the following fields:

Associated Form - Name of the associated form.

Form Name - You can select the Form name from this drop-down list. The pre-defined SKAWS_Users Form associated to the User list will also be available in this list.

Version - You can select the version from this drop-down list and default value is Latest Version.
