Saving the Process Diagram as Image

AVEVA Work Tasks allows you to save the process diagrams as images in different formats from the menu. In the earlier versions, you could right-click on the process diagram and save it as a BMP image.

  • You can select Save As EMF Image to save the diagram as an image in the EMF (Enhanced Meta File) format. This is a vector format, which is ideal for printing or embedding in MS Office applications.
  • You can select Save As PNG Image to save the diagram as a PNG image. This is useful when you need to capture the process diagram images for use on the web.

See also, Save As EMF Image and Save As PNG Image.
