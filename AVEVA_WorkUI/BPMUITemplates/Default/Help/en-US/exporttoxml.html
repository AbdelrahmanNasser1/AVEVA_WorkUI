Export to XML

You can export a process definition as an XML file. This makes the process design portable and accessible from other applications.

To export a process design as an XML file:

  1. From the Tools menu, select the Export option.
  2. In the Windows File Download dialog that opens, click the Save button.
  3. This opens the Windows Save As dialog, where you can click the Save button to save the XML file. In this dialog, you can browse to a new location for saving the file or change its name before saving it.
  4. In the File Download dialog you can click on the Open button to view the XML process definition.
