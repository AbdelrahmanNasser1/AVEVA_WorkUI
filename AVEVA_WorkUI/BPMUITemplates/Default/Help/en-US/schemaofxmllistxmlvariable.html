Schema of Microsoft SharePoint List/DocLibrary Xml Variable

The schema will be displayed based on the List or Doc Library settings. The schema of Microsoft SharePoint List/DocLibrary includes the schema of the List/Document, SystemFields and Attachments nodes.

Note: The schema of Microsoft SharePoint List/DocLibrary cannot be edited.

System Fields Nodes

System Fields nodes cannot be edited.

System fields

All the read-only fields will be displayed under the SystemFields category.
