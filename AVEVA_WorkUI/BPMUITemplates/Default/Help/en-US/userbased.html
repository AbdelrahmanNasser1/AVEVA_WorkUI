View Effective Security Rights for User/Role

You can search for a particular user and view the security group and its rights assigned to the user.

To view security and its rights for the role:

  1. In the Standard Settings tab, click View Effective Security Rights For User/Role. The Role Based and User Based tabs appear.
  2. Click User Based tab. The View Effective Security Rights For User/Role section appears.
  3. Select a user using the User Lookup dialog box.
  4. Click View Rights to view the effective security group and rights for the selected user.
  5. Click Security Snapshots to view the user name, roles mapped for the user along with priority, user mapping and effective security group for the selected role.
