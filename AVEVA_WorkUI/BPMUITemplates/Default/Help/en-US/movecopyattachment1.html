Move Copy: List Attachments

Use this property to enter names of attachments to move/copy specified attachments alone.

If the Include List Attachments property is set to Yes, all attachments will be moved or copied if no file name is specified.

This property is optional, and can be set if required.

Note: Either a source list or Physical/Network location must be specified.

Example: SampleFile.txt; SampleFile2.txt
