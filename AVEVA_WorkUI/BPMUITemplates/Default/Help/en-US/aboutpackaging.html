About Package

A proper understanding of the Package interface allows you to use the various features effectively. The Package interface consists of the following components:

  • Package Template: Use to design a Package, define artifacts to the template, and complete the entire process of designing and building a Package to exporting the package file.
  • Manage Package: Use to import packages that are exported from the Package Template of source environment, and install it in the target environment. 

    Note: Access to the above mentioned functionalities is based on the rights given to the users. The functionality will either be disabled or it will not be visible if the user does not have rights to execute them.
