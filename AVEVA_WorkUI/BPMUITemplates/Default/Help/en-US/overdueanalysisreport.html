Overdue Analysis Report

The Overdue Analysis Report in the BAM Dashboard is displayed at two levels:

  • KPI(s): The trend analysis for each KPI defined in the BAM Configuration is provided as a pie chart and bar graph.
  • Overdue Analysis: The Overdue Analysis report shows a graphical view of the pending workflows against the threshold.

Overdue Analysis

This graphical view shows how the pending workflow count stacks up against the prescribed thresholds in each time slot as defined.

KPI Analysis

The bar chart view shows the KPI against the threshold defined for it.
