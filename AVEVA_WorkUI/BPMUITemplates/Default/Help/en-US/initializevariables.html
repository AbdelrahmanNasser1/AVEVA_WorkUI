Initialize Variables

As soon as you create a variable, you have the option to initialize it.

You can update the variable values using the Update Variable activity, if needed. The supported types for initialization are:

  • Variables
  • Content variables
  • XML variables

Note:
You can also use the Expression Editor to initialize variables.
For number data types, the minimum and maximum values are not validated. Enter only valid numeric values for number data types.
For DateTime, Date, or Time data types, ensure that the date or time input format is based on the culture set. You can enter the input or use the Date and Time picker to enter the required date and time.
Date.CurrentDate is not supported. Use the tree view option to pick the required date and time.
The time format supports long data type to enable you to enter seconds.
