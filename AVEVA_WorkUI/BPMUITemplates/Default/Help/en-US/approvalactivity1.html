Approval Activity
  1. Select the AVEVA Work Tasks mail which has the Approval activity from the Microsoft Office Outlook Inbox.
  2. Right-click the mail to get the context menu.
  3. If the mail is not AVEVA Work Tasks mail, then the user will not get any context menu.
  4. Approval email options are Approve and Reject.
  5. When a reply is made, the mail icon changes to reply icon to notify the user that the mail is already replied.
  6. Once the mail is sent, the workflow engine will generate a confirmation mail.
  7. If an approval mail is sent again, a duplicate confirmation mail is received.

    Note:

    - Approval mail actions can be localized.

    - In [AVEVA Work Tasks Installed Path]\Locales\en-US\EmailTemplate.txt file, change the value for Email_ApproveActivity:Approve and Email_RejectActivity:Reject. This will reflect in the approval mail option1, context menu, and ribbon bar.
