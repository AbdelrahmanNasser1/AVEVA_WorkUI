Wait for Email: Set Variables And Contents from Email

Set Variables And Contents from Email Property window

You can set the filter expression to filter the incoming email in this property window. To open this window, click the button for the 'Set Variables And Contents from Email' property.

The window has two tabs at the top to manage the rows of variable settings in the grid.

  • New Row - You can click this tab to add the newly entered variable/content settings and display a new row to the grid where you can add another setting
  • Remove Selected - You can click this tab to remove the variable/content selected in the grid
  • Generate RegExp for Selected Row - You can click this tab to generate a regular expression for the search criteria. This displays the fields shown in the bottom half of the property window image. These fields are described later in the topic.

The grid has the following parameters that you need to specify for the new variable.

  • Variable - You can select the variable, to which you wish to assign the data in the attachment, from the drop down for this field.
  • Search criteria (regular expression) - You can enter a regular expression in this field to be used to search the email for data to be assigned to the variable/ content. Alternatively, you can click on the Generate RegExp for Selected Row tab to generate a regular expression.
  • Search in - You can specify which part of the email should be searched by selecting an appropriate option from the drop down for this field. You can set the activity to search in the entire email, the subject, and the attachments among other options.
  • Set variable in the named group - You can enter the name of a group, to which you can add the variable as a member, in this field.

Tip: You can specify "*" in the Search Criteria field to retrieve all the data.

When you click on the Generate RegExp for Selected Row it displays the following controls to generate the regular expression.

  • Start String - You can specify the start string for the regular expression in this field.
  • Delimiter - Start - You can enter the start delimiter for the regular expression in this field.
  • Delimiter - End - You can enter the end delimiter for the regular expression in this field.
  • Generate - You can click this button, after specifying the above values, to generate the regular expression.
  • Hide - You can click this button to hide these fields.

There are two buttons at the bottom of the screen.

  • Update - You can click this button to save the newly added variable/content settings.
  • Cancel - You can click this to close the property window without saving the newly added settings.
