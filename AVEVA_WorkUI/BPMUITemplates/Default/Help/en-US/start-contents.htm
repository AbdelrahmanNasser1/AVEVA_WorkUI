Contents Property window

You can set the workflow contents in the Contents property window. To open this dialog box, click the button in the 'Contents' property. The contents are displayed in a grid, which also has a blank row where you can enter the parameters for a new content.

The dialog box has two tabs at the top to manage the rows of contents in the grid.

  • Add Row - You can click this tab to add the newly entered content and display a new row to the grid where you can add another content.
  • Remove Selected - You can click this tab to remove the selected content row in the grid.

The grid has the following parameters that you need to specify for the new content.

  • Content Description - You can use this field to enter the content description.
  • Type - You can use the drop down in this field to select the data type for the variable.
  • XPath Expression - You can enter the XPath expression for the content in this field.
  • Attribute Name Blank for InnerString - You can enter the attribute name in this field to get the XML attribute. If left blank, it will retrieve the XML value.

There are two buttons and a check box at the bottom of the dialog.

  • Update - You can click this button to save the newly added content(s).
  • Close - You can click this to close the property window without saving the newly added contents.
  • Last Row needs to be added - You need to select this check box to save the last row added.
