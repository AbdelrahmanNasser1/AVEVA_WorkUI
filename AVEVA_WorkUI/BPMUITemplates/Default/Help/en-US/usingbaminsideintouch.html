Using Workflow Status View Inside InTouch

Follow Step 1 to Step 7 listed in the Initial Steps and continue:

  1. Select the control in the editor.

    The Edit Animations window appears.

  2. In the Animations pane, double-click and open Workflow status Configuration.

    A new configuration page appears. You can configure the Control in this page.

  3. After configuring the Workflow Status, click OK.
  4. Save and close the editor.

  5. Select the instance and deploy in the ArchestrA IDE.
  6. Double click the $InTouchViewApp instance (aaReactorDemo) under ViewEngine from the ArchestrA IDE.

    The InTouch WindowMaker window appears.

  7. Create a new window for the Workflow Status View control.

  8. Provide the name and size for the window.

  9. Right-click inside the newly opened window and select "Embed ArchestrA Graphic" option.

    The ArchestrA Galaxy browser appears.

  10. Select the Instances tab, and select the object to which the graphic symbol has been added.

  11. Select the name of the graphic object (Workflow Status View) that has to be displayed.

  12. Enable security for the Workflow window, if required.

  13. Save the window, and exit InTouch WindowMaker.
  14. Save and close the object in the ArchestrA IDE.
  15. Deploy the $InTouchViewApp instance.
  16. Open the InTouch Viewer.

  17. Click Special, click Security, and then click Log On.

    The login dialog box appears.

  18. Enter the Credentials and click OK.

  19. Open the created Workflow Status View window from the file menu.

    The Workflow Status View Control loads in the window.
