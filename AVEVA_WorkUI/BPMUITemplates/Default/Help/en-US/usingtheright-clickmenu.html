Adding Activities Using the Context Menu

An alternative method to add activities to the design mosaic is to use the context menu option.

To add an activity in this way:

  1. Select an activity on the design mosaic with which you want to link the new activity.
  2. Right-click the icon for the activity to be added to open the context menu shown below.
  3. From the menu, select the appropriate Add Activity option. This will add the new activity to the design mosaic and link it to the selected activity.  
     

Tip: You can specify the position of the new activity relative to any activity that is currently selected on the design mosaic using one of the relevant options in the menu. For example, you can select the "Add above selected Activity" option to add the new activity icon above the selected activity.

You can replace the selected activity with the new one by selecting the "Replace selected Activity" option. You can also set link options in the context menu. The "Link from Selected Activity" option will link the new activity from the selected activity. The "Link to Selected Activity" option will link the new activity to the selected activity. Select the "Don't Link Automatically" option if you do not want the new activity to be linked automatically to the selected activity. In this case, you can link the new activity manually to any activity on the design mosaic.
