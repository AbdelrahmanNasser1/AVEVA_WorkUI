View Effective Security Rights for Roles

You can search for a role to view the security group and rights assigned to it.

To view security and its rights for the role:

  1. In the Standard Settings tab, click View Effective Security Rights For User/Role. The Role Based and User Based tabs appear.
  2. Click Role Based tab. The View Effective Security Rights For Role section appears.
  3. Select a role using the Role Lookup dialog box.
  4. Click View Rights to view the effective security group and rights for the selected role.
  5. Click Security Snapshots to view the role name along with the security group to which the selected role belongs.
