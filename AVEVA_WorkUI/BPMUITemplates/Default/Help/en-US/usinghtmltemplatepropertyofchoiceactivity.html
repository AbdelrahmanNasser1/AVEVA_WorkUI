Using 'HTML Template' Property of Choice Activity

You can use the html email template that is present in the physical folder ..\AVEVA\Work Tasks\WorkflowElements\Default\en-US\Templates\OutlookEmailTemplate\Choice.html in the 'HTML Template' property of Choice activity.

Complete the following steps to customize the Choice.html.

  1. Right-click the Choice activity and select Activity Properties.
  2. Under Mail Channel group, set the Use html template from property to Yes.
  3. Under Mail Channel group, click the HTML template edit icon.
  4. Go to ..\AVEVA\Work Tasks\WorkflowElements\Default\en-US\Templates\OutlookEmailTemplate\Choice.html path and take a copy of the default html template file of Choice activity and make the necessary changes to the copied file.

    Note: While modifying the Choice email template the following place holders should not be modified as these are necessary for the AVEVA Work Tasks engine to process the mails that are received from the actors.
