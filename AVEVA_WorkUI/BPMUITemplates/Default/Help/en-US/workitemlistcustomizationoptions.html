Work Items List Layout Customization

The Work items list view provides list management features for sorting, grouping, column selection, filtering and rearranging. Right-clicking on any column in the column header bar brings up a menu with the available options.

List Management Options

A description of the available options follows:

  • Sort A to Z - Sorts the list on the selected column in ascending order
  • Sort Z to A - Sorts the list on the selected column in descending order
  • Group By This Field - Groups the list by the selected column
  • Group By Box - Displays the Group By box at the top. You can drag and drop a column header in this box to group the list by that field. This is an alternative way to group the list.
  • Remove This Column - Removes the selected column.  The removed column can be restored by reselecting the column from the Select Columns option.
  • Best Fit - Changes the column width to best fit
  • Filter Bar - Toggles the filter bar from on and off
  • Select Columns - Provides a selection list with all available column headers. Columns displayed in the view are checked. Click Apply Changes for the new settings to take effect. Click Reset to revert to the previous settings. You can also rearrange the columns by dragging and dropping them to a new position.
