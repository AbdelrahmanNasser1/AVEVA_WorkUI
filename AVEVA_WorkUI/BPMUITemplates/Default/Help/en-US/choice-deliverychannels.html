Choice - Delivery Channels

Purpose

The purpose of this chapter is to describe the feature of WorkflowStatus Form of Delivery Channels property under the category Configuration of any human activity available in AVEVA Work Tasks.

Description: This feature enables us to view the status of the workflow in the work item list in the graphical form. [Optional]

  1. Select the Human Activity.
  2. Click the Delivery Channels Property.

    This will open a window like the one shown below.

  3. Right-click WebWorkList to display the context menu with the Add Instance of Workflowstatus option as shown below.

  4. Set the Properties for the Execution View Status Graphical view.
  5. Click Save.
  6. If you want to set the workflow status form as the default instance, right-click WorkflowStatus1, and select the Set as default instance option from the context menu.

  7. Click Save and Save Channels.

Output:

Now, when you click on any row in the Work items list, you will find this Execution view page being displayed as the detailed view if you set it as a default instance.

Otherwise, the workflow status form will appear in the context menu of each row in the Work items list and you can open the execution view by selecting it from the context menu.
