Using Queue Control to Create and Manage Queues

You can add and define queue from the Queue Management page.

To add and manage a new queue:

  1. In the Enterprise Console, click the Menu button, click Settings, and then click Queue Management. The Queue Management page appears.
  2. Click New Queue to add a new queue.
  3. In the Name box, type the unique name for the queue.
    Ensure that the Queue Name does not contain more than 250 characters, and does not contain the following special characters: [#~&+\"\'<>|@!$%*()=/{}?]\
  4. In the Description box, specify the description for the queue.
    Ensure that the Queue Description does not contain more than 150 characters.
  5. Click Add. The Queue is added and a new page appears that allows you to manage participants by adding the Users or Roles for the queue and setting their access rights. You can allow the users to view work items, pick up one, or pick the next available task without being able to view it, or put back a task in the queue.

You need to specify the Queue settings, set dispatch rules, and set escalations using the appropriate tabs in this window. For more information, refer to Queue Management.
