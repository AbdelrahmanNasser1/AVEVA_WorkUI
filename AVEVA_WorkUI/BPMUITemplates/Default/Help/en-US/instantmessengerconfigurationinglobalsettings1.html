Instant Messenger Configuration in Global Settings

The Jabber User Id and Password can be stored in the Communication tab of Global Settings window. This ensures that the User Name and Password is updated automatically in the Notification/Registration window while configuring the Notification Activity with Notification Type as IM.

The section below gives step by step instructions wherein you can store the Jabber User Id and Password

  1. Navigate to the Global Settings window of AVEVA Work Tasks Farm Configuration.
  2. In the Communication tab, IM Settings section enter the User Name and Password of the Jabber account.
