Method 1: Right Click and Select Activity Properties

In AVEVA Work Tasks, you only need to open the Properties area when you are going to set the properties for the selected activity. This is in contrast to the earlier versions, where the Activity Properties area was always displayed, even if the properties had already been set. This change results in more space in the Process Designer UI for the design mosaic.

You can open the Activity Properties area for a selected activity by right-clicking the activity icon and selecting the Activity Properties option from the context menu. This is one of the three ways in which you can access the Activity Properties area.

  1. Right-click the icon of the activity for which you need to set the properties.
  2. In the context menu, select Activity Properties, which is the first option.
  3. This will display the Activity Properties area, with all the properties of the selected activity. You can set the required properties and click Save to close the Activity Properties area.
