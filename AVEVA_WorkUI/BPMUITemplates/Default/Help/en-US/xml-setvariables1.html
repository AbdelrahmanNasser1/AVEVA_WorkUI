XML Action: Set Variables

Set Variables Property window

You can set Xpath expressions to workflow variables in this property window. To open this window, click the button for the 'Set Variables' property. It displays the settings for the variables in a grid, which also has a blank row where you can enter the settings for a new variable.

The window has two tabs at the top to manage the rows of variable settings in the grid.

  • Add Row - You can click this tab to add the newly entered variable settings and display a new row to the grid where you can add another variable settings.
  • Remove Selected - You can click this tab to remove the variable selected in the grid.

The grid has the following parameters that you need to specify for the new variable.

  • Variable Description - You can select the variable to which to assign the XML data from the drop down for this field.
  • Type - You can select the variable type from this drop down.
  • XPath Expression - You can use this field to enter the XPath Expression for the XML data to be assigned to the selected variable.

There are two buttons and a check box at the bottom of the screen.

  • Update - You can click this button to save the newly added variable settings.
  • Close - You can click this to close the property window without saving the newly added variable settings.
  • Last Row needs to be added - You need to select this check box to save the last row added.
