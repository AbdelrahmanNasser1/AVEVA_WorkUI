Showing of Specific Workflow Instances in BAM Control

BAM is used for viewing all the workflow instances in different categories like Running Workflows, Failed Workflows, Aborted Workflows, Paused Workflows and Completed Workflows. This feature allows you to filter and show only specific workflow instances in the BAM control.

Showing the specific instances of the workflow in BAM is controlled by the property – InstancesToBeShown. This property is an ArrayList property to which you can add the Executionids of the workflow instances for which you want to see the details.

The property InstancesToBeShown is set say for the Execution ids 25, 24, 23, 22, 21. The report will be shown only for these workflow instances.
