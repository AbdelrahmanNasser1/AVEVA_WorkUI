Forms Web Part

You can set the display properties for the Form List Web Part in the Custom Properties section of the web part.

  1. On the Settings menu, click Edit Page.
  2. Click Add a Web Part.
  3. In the page that appears, select AVEVA Work Tasks under Categories, and AVEVA Work Tasks Forms under Parts.
  4. Click Add.
  5. Click Edit Web Part from the edit options of AVEVA Work Tasks Forms web part.
  6. In the AVEVA Work Tasks Forms property window, expand Custom Properties.
  7. The Forms List Web Part has the following properties that you can use to select the display of the Forms List Web Part.
    • Form Display Mode: By default, Form Display mode property is set as mode.
    • Show Fill form as popup: Use this option to view the Fill Form as a pop-up on the screen.
    • Tree View Height: Use this option to specify the height of the tree view of the web part.

Note: To access the Forms Web Part page, you must have Can Fill Form permissions in AVEVA Work Tasks.

For more information about AVEVA Work Tasks Web Part properties, see AVEVA Work Tasks Out-of-the-Box Web Parts section in the User Guide.
