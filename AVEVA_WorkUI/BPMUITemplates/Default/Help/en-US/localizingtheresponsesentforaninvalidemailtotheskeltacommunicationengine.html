Localizing the Response Sent for an Invalid Email to the Communication Engine

The response sent for an invalid email, for the Human Activities email or for the WaitForEmail Activity, to the AVEVA Work Tasks Communication Engine, can be localized. The value of InvalidFilterMailMessage key in the channel.txt file is emailed as a response to any invalid email. The channel.txt file can be found at  <AVEVA Work Tasks Install Directory>/Locales/en-US.

The (key, value) pair can be specified in the channel.txt file as follows:

InvalidFilterMailMessage: <Localized message>

A sample of the value for this key is shown below

"System could not process your mail. Some of the key words not found in your mail to process it. Reply to the original message again."
