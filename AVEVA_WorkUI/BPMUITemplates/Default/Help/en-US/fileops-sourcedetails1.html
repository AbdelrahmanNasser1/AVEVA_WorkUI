File Operations: Source Details

Source Details Property Window

You can specify the details of the file source in this property window. To open this window, click Source Details property. The Source Details property window appears as shown in the following image:

The following table shows the fields of the Source Details property window:

Parameter

Description

Protocol

The required file transfer protocol, such as FTP-HTTP.

Location

The URI or UNC of the file or directory.

Pattern

Pattern to decide the file types to be transferred, such as *.doc,*.xls.

UserName

User name for the source.

Password

Password for the source.

Domain

Domain or machine of the source name.

PortNumber

Port number of the source server.

Each of the above parameter fields has the following controls.

  • Operator: Select the operator for the expression from the drop-down menu. For example, "="
  • Value: Select the type of value to assign to the parameter in the expression. You can specify Variable, Content, Value, or XMLVariables here.
  • Value text box or drop-down menu: Specify the actual value to assign to the parameter. The control displayed depends on the option selected in the Value column. If you select "Value", it will display a text box here. You can enter a discrete value in this box to complete the expression. If you select "Variable", "Content", or "XMLVariables", there will be a drop-down menu instead of the field. This drop-down menu contains the variable collection, or content collection, or XMLVariables collection from which you can select an appropriate option.
  • Submit: Click Submit to save the specified parameters.
  • Cancel: Click Cancel to cancel the settings and close the property window.

    Note: In AVEVA Work Tasks, the validation for user credentials for FTP-HTTP is not provided. However, if a particular FTP needs authentication, then the user has to provide appropriate credentials.
