How to Use Static Members of the Class?

This example shows how to use static members of the class in SOA Property Editor/Expression Editor. To use Static members, there is no need to create the instance of the class. After registering an assembly into the SOA Assembly list, all the static members of the registered class will get loaded into the SOA Property Editor/Expression editor and can make use of the same for passing the parameter.
