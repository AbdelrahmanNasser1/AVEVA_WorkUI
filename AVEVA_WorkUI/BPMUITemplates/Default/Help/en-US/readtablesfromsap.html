Steps to Read Tables from SAP

To read the data from SAP Tables, follow the steps given:

  1. Create the XMLVariable of type XML SAPTable. See SAPTable Navigator Activity - XML Variable of SAPTable Type for creating XMLVariable of type XML SAPTable.
  2. Select the table from the table list that will store the information inside  XMLVariable of type XML SAPTable.
  3. Drag and drop the SAPTable Navigator activity in Process Designer.
  4. Create a query using SAPTable Query property. See SAPTable Navigator Activity - SAPTable Query for more details.
  5. Iterate through the records in the table using ForEach-Next activity.
