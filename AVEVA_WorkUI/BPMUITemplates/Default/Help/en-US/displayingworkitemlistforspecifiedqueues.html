Displaying Work Items List for Specified Queues

The property ShowQueueNames enables you to display only specified queue folders. For example, if work items are assigned to Finance queue and Purchase Department queue, you can create two controls for Work items list with the first set to display only the Finance queue folder and the other set to display only the Purchase Department queue folder.

The display of queue folder is subject to the participants right to View Queue Items for the queue. For more information on participant rights, see Participants Tab.
