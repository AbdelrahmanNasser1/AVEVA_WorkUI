Showing Custom Link in Work Item List Control

This feature enables you to create custom links in the left hand side tree view menu of the Work items list control.

Scenario

John is the manager in Telematics organization acting on the HR and Accounts Payable Process. He wants to view a custom page like the Workflows list page from his work items Inbox on click of a link from the LHS tree menu view of Work items list control. He also wants this custom link to be secure so that only the users with proper rights should be able to view it. He also wants sub links under the main custom link with different navigations.

You can create custom links through the Navigation List of Repository Settings.

Whenever a new repository is created, an entry - ec_menu_WorkitemCustomLinks is created in the Navigation List. You can create sub links under this Navigation link or can edit the link for adding a page if no sub link is needed.
