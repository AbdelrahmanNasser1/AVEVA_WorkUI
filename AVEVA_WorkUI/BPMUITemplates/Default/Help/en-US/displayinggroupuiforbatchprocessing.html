Enabling Multi-Select Option in Work Item List

When the DisplayGroupUI property is enabled, an additional button Enable Multi Select is displayed in the Work items list. This property is a powerful feature that expedites processing of similar items in a batch.

The Enable Multi Select button provides users with options of acting on items as a batch. An additional option is made available when the items are grouped.

Enable Multi Select Option in Work items list

Note: The Multi Select option is not supported for Resource Set activity, Information activity, and InvokeForm activity.

Work Item List in Multi Select Mode

On clicking the Enable Multi Select button, the Work items list interface is displayed in the multi-select mode with check boxes for each item. The user can enter comments that are applied to all the selections. The options for acting upon the selection depend on the type and stage of processing that the selected items are in.

Example 1: A user has multiple new work items that have the subject Approval of Vendor Invoices. The user can enable multi-select, then select all the items that he wants to take the ownership of. He can then select the Take Ownership option in the drop-down to take the ownership of the selected items in a single click.
