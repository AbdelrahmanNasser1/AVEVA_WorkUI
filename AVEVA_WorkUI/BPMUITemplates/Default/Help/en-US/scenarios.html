  1. Create an XML Variable for the list with scope New.

  2. Design the process.

  3. Update the variable activity.

  4. Create a folder called New Folder under root folder of the list.

  5. To create a folder under the folder New Folder, update the in list navigator folder name property with New Folder.

  6. Check if the folder exists. In the following workflow, find out whether New Folder exists in a list or not.

  7. Build a CAML query. The following query returns an item if the folder exists in the list.

  8. In decision activity, build the condition as follows. If the preceding query returns an item, then that item will be compared with the value using decision activity.
