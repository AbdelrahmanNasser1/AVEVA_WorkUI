Configuring Resource Calendar by Administrator

Users with administrative rights can configure the resource calendar for other users.

  1. In the Enterprise Console, click the Menu button, click Settings, and then click Calendar.

    The Calendar Control page appears.

    One global calendar should be defined to set the Resource Calendar.

  2. Click Set Resource Calendar.
  3. Select the required provider from the Provider drop down.
  4. Select the user for whom the Resource Calendar is to be set in the User Lookup pop-up.

    The following image shows user lookup with the selected resource:

  5. Calendar Control will be displayed for the selected user.

  6. Save the calendar and then set the work slots.

Note: The right Can Add Resource Calendar is added to the Calendar Control Rights to set whether a user can configure Resource Calendars or not. By default, the administrator will have the Can Add Resource Calendar right checked.
