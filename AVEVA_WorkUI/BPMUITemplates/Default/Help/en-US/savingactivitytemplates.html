Activity Templates

The sequence of steps provided through AVEVA Work Tasks Wizards can be saved as a template. You can use the activity templates to save a common set of actions that can be applied to multiple processes as a template. For example, if all routine HR tasks require the HR Manager's approval, this process can be saved as a template and used whenever required. The option to save the wizard as a template appears as the last step.

To save the activity as a template:

  1. Save process as a new template: Select Yes to save the template.
  2. Template Name: Enter the template name. Template names must not have spaces.
  3. Display Name: Enter the display name.
  4. Template Description: Enter the template description.

Templates can be used in other wizards. The Link to an existing template option is displayed in the Define Actions window as the path that the workflow must take after completion of the activity being setup.
