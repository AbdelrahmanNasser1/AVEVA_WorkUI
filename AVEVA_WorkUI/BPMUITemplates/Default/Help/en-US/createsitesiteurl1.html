Create Site: Url Name

The URL Name is a string that contains the new website URL relative to the root website in the site collection.

In this window, enter only the site name that should be appended to the site. You do not need to add the complete URL. This is a mandatory property.

In the example shown in the following image, do not add the URL. You must enter only MySite.
