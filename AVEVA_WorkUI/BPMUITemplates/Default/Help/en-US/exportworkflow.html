Exporting a Workflow

You can export a workflow definition as an XML file. This makes the workflow portable and accessible from other applications.

To export a workflow as an XML file:

  1. Select and right-click the workflow that you want to export from the workflow list.
  2. Click Export Workflow. The Export dialog box appears.
  3. Select the workflow definition which you want to export.
  4. Click Export. Save the XML file.
