Searching for Content in Workflow

AVEVA Work Tasks has a provision to search for content in the workflow. This is a useful tool to navigate large and complex workflow process diagrams.

To find a particular activity in a large process diagram:

  1. Enter the name or part of the name of the activity in the Search text box.
  2. Click the magnifying glass icon to start the search.

The search will display the activity that matches the search string on the design mosaic.
