Auto-ownership on Row Click

The property AllowAutoOwnershipOnRowClick enables you to allow users to take the ownership of a work item by clicking on the item in the Work items list. If this property is set to true, the option Take Ownership in the Manage Work Items field in Work Item Details will not be available.


Setting this property to true is useful in scenarios that require strict adherence to performance indicators, which measure the time when a work item is assigned to the time when a user takes ownership of it.
