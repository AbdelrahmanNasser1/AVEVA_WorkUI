Work Items List Manager View

The Manager View folder displays a list of all work items of all users reporting to the logged in user. This view enables managers to quickly access the status of  the work items assigned to reporters. Work item details are not displayed in this view.

Manager View Options

The following options are available from the right-click menu for an item:

  • Reassign Work Item - Enables work items to be reassigned to a different user. Reassigning the work item will remove it from the current user's work item list and transfer the item to the user to whom it has been reassigned.
    Once the work item is reassigned to a different user, the pop-up window will be closed automatically and the user will not be able to act upon the work item by reopening it.
  • Delegate Work Item - Enables work items to be delegated to a different actor. When a work item is delegated, it continues to appear in the list of the original user.
