Reusing Existing Process Designs by Renaming Them

AVEVA Work Tasks allows you to use an existing process diagram as the basis for creating new process designs. In scenarios where there are a number of similar process designs, this helps reduce the time required to design all the processes. It will be enough to design one process and reuse its diagram for creating all the other process diagrams.

To reuse an existing process diagram:

  1. Open the process design file to be reused.
  2. From the File menu, select the Rename File option.
  3. Specify the new name for the process design file in the window that opens.
  4. Proceed to modify the process diagram as required and save the modified process diagram.
