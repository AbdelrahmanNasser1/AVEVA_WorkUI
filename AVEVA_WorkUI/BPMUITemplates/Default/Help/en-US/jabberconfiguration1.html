Introduction to Jabber Configuration

The Notification Activity is incorporated in the AVEVA Work Tasks workflow process designer to send notifications to actors, in the form of email, instant messages, and SMS. Jabber technology is being used to send instant messages. Jabber.org is an IM service, based on the open standard for instant messaging.

This document gives detailed instructions on registering a MSN Account with the Jabber server (Ex: Jaim.at, Jabbernet.dk) using Psi.  Psi is an instant messaging application that connects to the free and open Jabber IM network.

This document also explains the process of configuring the Notification activity to send instant messages from the AVEVA Work Tasks workflow using AVEVA Work Tasks list as the user provider.
