Using Queues in Workflows

In the Activity Properties pane of all activities, which require human intervention, there is an option called Assign Queue(s). You can use this property to specify a queue to which the corresponding work item will be assigned.


To assign a queue for an activity:

  1. In the Process Designer for Workflow, right-click the resource activity, and then select Activity Properties.
  2. In the Activity Properties pane, click the button for the Assign Queue(s) property.
  3. In the window that opens, select the queue(s) from the Available Queues list box.

    You can select multiple queues by selecting with the <Ctrl> key pressed.

  4. After selecting the required queue(s) click the [>] button to move it to the Selected Queues list box. You can click the [>>] button to select all the queue(s) and move them to the Selected Queues list box.
  5. You can remove any queue from the selected list. Selecting the queue and then click the [<] button. You can remove all the queues by clicking the [<<] button.
  6. After adding the required queue(s), click the Assign button to assign the activity to the selection.
  7. Click the Close button to close the window without saving the selection.


Difference between direct task assignment and task assignment through queue:

A task can get assigned to multiple resources directly whereas only one resource will get the task from a queue.
