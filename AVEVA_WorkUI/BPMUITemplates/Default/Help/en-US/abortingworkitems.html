Aborting Work Items

You can abort or kill work items in the Manage Work items window. To abort work items, the logged in user should have the permission to abort the work items. For this, the Can Kill Work Item right should be checked.

To abort a work item:

  1. In the Enterprise Console, click the Menu icon, click Tools, and then click Manage Work items. The Manage Work items page appears.
    All work items appear in a grid showing the information about work items assigned to different users within the application.
    The Manage Work items is visible only to users who have the right to forward work items.
  2. Select the work item(s) that you want to abort. You can select multiple work items at a time by selecting the check box in the grid.
  3. Click Abort. Alternatively, right-click the work item, and then select Abort. A confirmation message to abort the work item appears.
  4. Click Yes to confirm the abort, or No to cancel the abort. The message about successful abort appears.
  5. Click OK.
