Creating XML Variable of Type 'XML SAPTable'

To create the XMLVariable of type XML SAPTable in Start activity, follow the steps given below:

  • Click New Variable button in the XML Variable User Interface.
  • Enter the following information:

    Variable Name - Enter the name of the variable.

    Description - Enter a description for the variable.

    Type - This field defaults to XML Document. Select XML SAPTable from drop-down.

    Scope - Select the scope of the variable. The scope of the variable can be either Global or Local.

  • When the variable type is changed to XML SAPTable, two additional properties are displayed as seen in the image below.

SAP Application Server: The name of the host that you want to connect to the SAP system. Click the  icon to open SAPTable List.

Listing SAP Tables

To display the tables from SAP, follow the steps given below:

  1. In the Connections drop-down, select specific connection.
  2. In the No. of Hits text box, enter the number of tables to be displayed. By default, it displays ten tables.
  3. Click Search to display SAP Tables. If no search criteria is mentioned and No. of Hits is not given, then the list of ten randomly selected tables will be displayed.
  4. Select table name, and then click Select to generate the XML Schema. The XML variable of type XML SAPTable is generated with schema, SAP server information, and table name.

Advance Search

SAP tables are searched based on the criteria provided in Advance Search.

Example:

Select the connection from the connection list.

Enter No. of Hits = 10.

In Advance Search, enter search criteria = sales%, select Table description from drop-down. It will display tables whose description starts from 'sales'.
