BAM Rule Settings

To create a rule for pending Workflow (Overdue):

  1. In the Alert Configuration tab, click Rule to create rules for both Overdue and KPI. By default, the Rule Settings window appears.
  2. Check Total Workflow count is, and then select GreaterThan.
  3. Check Count in each slot is, and then select GreaterThan.
  4. Click Select Slots to set the rule for slots. The Select Slot dialog box appears.
  5. Select the slot, and then click OK.

Creating rule for specific type of work (KPI)

  1. Check Count in each datapoints of KPI(s) is, and then select GreaterThan.
  2. Click Select KPI(s) to set the rule for KPI(s) that need to be monitored. The Select KPI(s) dialog box appears.
  3. Select the KPI, and then click OK.
  4. Click Save. A message about successfully creating alert appears.
  5. Click OK. This alert is triggered when the BAM report is being processed.
