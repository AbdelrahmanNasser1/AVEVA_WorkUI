Setting an Approval Status for an Item

AVEVA Work Tasks allows you to set approval status for an item using XMLVariables of type SP List/DocLibrary. List Navigator activity and List Operations activity can be used to set the status.

To set the status for an item, you need to select the Set Approval Status option from Operations property and select Approved from Approval Status property of List Operations activity to set the status.

Note: You can also set status as Denied or Pending’from Approval Status property.
