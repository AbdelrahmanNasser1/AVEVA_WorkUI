Defining Layouts at Workflow Level

The layout for the Work items list can be defined for a Workflow in the Process Designer by selecting the Work item list Layout option from the Settings menu. Selecting this option brings up the WorkList Layout Customization window. The Worklist Layout can be customized for the following:

  • Column positions
  • Column selection
  • Grouping a column
  • Sorting a column

If you set the ShowInboxViewonly property to true, Workflow layouts will not be applied.

The default view of Work items list in the Enterprise Console cannot be customized. The above customization applies only to work item lists created by embedding the Work Item List Control in an ASPX page. For more information about using AVEVA Work Tasks Controls as a part of a project, see Using AVEVA Work Tasks Controls in Applications in the Developer Guide.
