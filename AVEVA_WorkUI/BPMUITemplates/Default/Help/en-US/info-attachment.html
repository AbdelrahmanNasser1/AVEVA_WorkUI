Info - Attach Reports

Configuration of Mail Attachment property window

You can configure the Attachment property for the Approval activity in this property window.

In this property window, you can configure the report attachments to be emailed.

  • Attachment Type: Use this property to select the type of attachment for sending the report as an email.
    • Email attachment: Select this type to send the report as an email attachment.
    • Link to report: Select this type to embed a link in the email to download the configured report.

  • Attach Report: Use this property to attach a report generated by the Report Generator activity within the workflow or to select a report from the list of generated reports.

    You can attach a report by selecting the required Report Reference Variable using the Expression Editor. Alternatively, you can select the report from the Reports List window which consists of the already generated reports.

There must be at least one Report Generator activity before an Approval activity in the workflow to select the Report Reference Variable (using Expression Editor) as an Attachment. The Report Reference Variable is generated while configuring the Report Generator activity. By default, the report reference variable is [Template Name]_[activity Name]. For more information about Report Reference Variable, see Configure Report Template.

To configure mail attachment:

  1. Click the edit button for the Attachment property.
  2. Select the required Attachment Type.
  3. In the Attach Report text box, you can either use the expression editor to get the variable of the report or use the Reports List to select the report.
  4. To get the variable, click the Expression Editor icon given beside the text box to select the required Report Reference Variable.
  5. In the Expression Editor window, select the required variable.

  6. Click OK.
  7. Click Close.

    The variable appears in the Attach Report text box.

  8. To select the report from the Reports List dialog box, click the Reports icon. 

     

  9. In the Reports List dialog box, select the required report, and then click Submit.

    This window closes automatically.

  10. Click Close to close the configuration window.
