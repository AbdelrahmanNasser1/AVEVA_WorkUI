Displaying Work Items List for Specified Workflows

The property WorkflowNames enables you to display work items assigned by specified workflows. For example, if work items are assigned by workflows for Invoice Processing and Expense claims, you can create two controls for Work items list with the first set to display only items assigned from Invoice Processing and the other set to display only Expense work items.

Note: This property is effective only when the DisplayLeftTreeMenuAtApplicationLevel property is set to False.
