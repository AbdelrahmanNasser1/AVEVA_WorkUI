Customizing Work Items List Layouts

The field layouts in Work items list can be defined at the following levels:

  • Global - Work items list layouts can be specified for the entire application programmatically.
  • Defining Layouts at Workflow Level - The layout for a Workflow can be specified in the Process Designer while designing the Workflow.
  • Defining Layouts for Queues - The layout for work items assigned to the queue can be specified in the Queue Management window.
  • User - Users can further personalize the work item list views and save layouts.

Global layouts are applied only to those workflows and queues that do not have Workflow/queue specific layouts defined and can be saved with a specific name.

The Work items list also enables filtering of data (see Filtering Data in Work Items List) within a list.
