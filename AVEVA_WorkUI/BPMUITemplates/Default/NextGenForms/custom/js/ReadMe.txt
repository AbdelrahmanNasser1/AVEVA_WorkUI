1. CUSTOM JAVASCRIPT

----------------------
1. CUSTOM JAVASCRIPT
----------------------

1. Use includeCustomJsFiles to include the javascript file(s).
2. Create a javascript in the <BPMUITemplates>/NextGenForms/custom/js/ folder.
3. Multiple Javascript can be included includeCustomJsFiles method 

Ex  : includeCustomJsFiles(['MyJS1.js','MyJS2.js']);
